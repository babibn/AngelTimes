# Angel Time JSON (ساعت‌های فرشته‌ای)

این مخزن شامل یک لیست JSON از «ساعت‌های فرشته‌ای» و ساعت‌های آینه‌ای در فرمت ۲۴ ساعته است که معادل آن‌ها در فرمت ۱۲ ساعته همراه با AM/PM هم در کنارشان آمده است. این زمان‌ها شامل عددهای تکراری مثل `11:11`، `2:22` و ساعت‌های آینه‌ای مثل `13:13`، `14:14` و غیره هستند.

## ساختار JSON

هر آیتم در آرایه‌ی JSON ساختار زیر را دارد:

```

{
"time24": "HH:MM",
"time12": "HH:MM AM/PM"
}

```

مثال:

```

{
"time24": "23:11",
"time12": "11:11 PM"
}

```

## فایل‌ها

- `angel-times.json` – فایل اصلی JSON که تمام ساعت‌های فرشته‌ای و آینه‌ای را دارد.
- `README.md` – مستندات انگلیسی.
- `README.fa.md` – همین فایل، مستندات فارسی.

## موارد استفاده

می‌توانید از این JSON در موارد زیر استفاده کنید:

- پروژه‌های JavaScript/TypeScript (فرانت‌اند یا بک‌اند)
- اپلیکیشن‌ها یا ویجت‌هایی که بر اساس ساعت واکنش نشان می‌دهند
- پروژه‌های مرتبط با عددهای فرشته‌ای و نمادشناسی اعداد
- هر پروژه‌ای که به یک لیست از ساعت‌های خاص روی ساعت دیجیتال نیاز دارد

### مثال در جاوااسکریپت

```

import angelTimes from "./angel-times.json" assert { type: "json" };

function isAngelTime24h(time24) {
return angelTimes.some((t) => t.time24 === time24);
}

console.log(isAngelTime24h("23:11")); // true

```
